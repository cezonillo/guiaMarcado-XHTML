Migas de pan

Las migas de pan, como todo el mundo sabe, se utilizan de forma muy habitual para indicar al usuario el lugar donde se encuentra dentro de la estructura de páginas de un portal.

De forma general, utilizaremos una capa que envuelva al camino, un span inicial si es necesario un texto introductorio (por ejemplo, "estas en"), y un listado de enlaces que finalice con un span que refleje la página actual.

Ejemplo visual

Estás en:
  1. Inicio
  2. Nivel 1
  3. Nivel 2
  4. Nivel actual
