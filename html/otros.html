Saltos de línea BR

Es un elemento HTML que inserta un salto de línea. Antiguamente su uso era muy común, pero hoy día, con un marcado correcto no debería hacer falta ya que una buena práctica si queremos insertar un salto de línea sería utilizar dos párrafos diferentes.

Ejemplo visual

Hola
¿Qué tal?

Código HTML

Observaciones

Es importante utilizar el código <br/> para los saltos de línea. Es muy habitual utilizar el código <br> lo que supone un error en el marcado XHTML, pues la etiqueta br quedaría abierta.
