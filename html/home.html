Guía de marcado: XHTML + CSS

Introducción

Esta guía pretende marcar una serie pautas a seguir a la hora de implementar el marcado de las aplicaciones Web desarrolladas por B2B 2000. De este modo se podrá conseguir una mayor eficiencia en el desarrollo y una mayor calidad en el marcado, utilizando un etiquetado correcto y coherente para cada tipología de contenido.

En el menú de la parte izquierda se encuentran cada uno de los componentes y tipologías de contenidos que se tratarán en la guía. Cada página contiene ejemplos visuales y el código HTML y CSS que los genera.

La guía nos servirá, por tanto, como un elemento de formación y/o apoyo para los desarrolladores y maquetadores de la empresa, y como herramienta para construir un marcado correcto, o entender mejor uno ya realizado.
