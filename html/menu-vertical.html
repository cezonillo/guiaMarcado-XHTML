Observaciones

Como se puede observar la opción 3 del menú tiene hijos. Estos podrán mostrarse inicialmente, o estar ocultos e ir desplegándose al seleccionar la opción 3 o pasar sobre ella. Todo ello se haría mediante técnicas CSS o javascript dependiendo de la funcionalidad requerida.

Es importante diferenciar con una clase la opción del menú seleccionada en cada caso (en el ejemplo se añade la clase "seleccionada").
