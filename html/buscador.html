Buscador

Se propone a continuación el buscador más común que consta de etiqueta, caja de texto y botón. Utilizaremos siempre esta estructura salvo en casos puntuales, ya que mediante estilos podremos hacer que un botón sea una imagen, ocultar la etiqueta, etc. en caso de que fuese necesario.

Con este código obtendremos un buscador con código HTML accesible.
