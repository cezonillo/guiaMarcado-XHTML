Elementos obsoletos

Aquí trataremos aquellos elementos y atributos obsoletos, es decir, que ya no deberían utilizarse en HTML.

Elementos HTML center, font, b, u, i

Estos elementos ya no se utilizan. Como sustitutos utilizaremos lo siguiente:

  • Para negritas y cursivas utilizaremos los elementos <strong> y <em> respectivamente.
  • Para cambiar el estilo de una fuente envolveremos el texto deseado en un span.
  • Para centrar algo se utilizaran las propiedades CSS necesarias (text-align:center en una capa, por ejemplo).

Atributos border, width, height, hspace, bgcolor, nowrap, color, nowrap, face, size

Estos atributos, utilizados generalmente en tablas, imágenes o inputs hemos de evitarlos pues estan obsoletos y muchos no son accesibles. Se dejan estas propiedades a los estilos CSS.
