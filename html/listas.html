Listas

La lista es uno de los elementos HTML más utilizado. Como su propio nombre indica sirve para presentar listados, pero además su uso es muy habitual para el marcado de menús y pestañas, mapas Web, etc.

Existen tres tipos de listas: listas ordenadas, listas desordenadas y listas de definición.

Además se muestra la forma correcta de marcar las listas anidadas

Observaciones

Pese a que los ejemplos mostrados presentan toda la información verticalmente, mediante los estilos correspondientes se puede indicar que los elementos del listado se sitúen horizontalmente (propiedad "display:inline" asociada al "li").
