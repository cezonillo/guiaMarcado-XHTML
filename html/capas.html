Capas contenedoras

Conocidas como capas o "div" se utilizarán para presentar información de forma entructurada y organizada. Se usarán tantas como bloques de información se consideren oportunos, tanto independientes, como anidados (es decir, bloques dentro de un mismo bloque).

En la mayoría de las ocasiones llevarán asociadas un identificador o clase, que permitirá darle estilos personalizados a cada una de ellas (anchos, imágenes de fondo, bordes, etc).

Observaciones

En el código de ejemplo se han utilizado siempre identificadores para las capas. En este caso no es ningún problema debido a que son únicos y solamente se utilizarán una vez por cada página. Sin embargo, es importante destacar, que si un tipo de bloque con sus estilos correspondientes se utiliza en más de una ocasión dentro del documento, utilizaremos el atributo "class" para aplicarle CSS, ya que no es semánticamente correcto que haya dos elementos con el mismo identificador.

Los "div" son elementos HTML en bloque, eso quiere decir, que por defecto ocupan toda un línea. En el apartado aspectos importantes se indican las diferencias entre elementos en bloque y en línea y las pautas a seguir a la hora de utilizarlos.
