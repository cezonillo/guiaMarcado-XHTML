Elemento SPAN

El elemento HTML "span" se utiliza de forma muy habitual para aplicar estilos CSS a partes concretas de un documento. Así, por ejemplo, si dentro de un párrafo queremos que una o varias palabras concretas tengan un color, tamaño o tipo de letra diferente. También resultan útiles para poder aplicar márgenes a un contenido en línea o hacer que un contenido en línea salte de línea. Una buena definición para el uso del span sería: "mételo en un span si quieres que sea diferente (en cuanto a estilos se refiere)".
